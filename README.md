# HaYaku

Mac 常駐翻訳アプリ。グローバルショートカットで選択テキストを OpenAI API で翻訳します。

## 機能

- メニューバー常駐（Dock アイコンなし）
- **⌘⇧T** で選択中のテキストを自動取得して翻訳（ショートカットは設定画面で変更可）
- 常に日本語に翻訳（入力が既に日本語ならそのまま返す）
- マウスカーソル近くにポップアップで結果を頭から逐次表示（ストリーミング）
- 設定画面で API キー・モデル・ショートカットを変更可能

## セットアップ

### 必要環境

- macOS 13 (Ventura) 以上
- Xcode Command Line Tools: `xcode-select --install`

### インストール

```bash
./install.sh
```

`/Applications/HaYaku.app` にビルド・インストールして起動します。

### 初回起動時の設定

1. メニューバーにバブルアイコン（💬）が表示されます
2. 設定画面が自動で開くので **OpenAI API キー** を入力して「保存」
3. アクセシビリティ権限ダイアログが出たら「システム設定を開く」→ HaYaku にチェック
4. アプリを**完全終了して再起動**
5. テキストを選択して **⌘⇧T** で翻訳開始

## API キーについて

API キーは以下に **平文で** 保存されます:

```
~/Library/Application Support/HaYaku/config.json
```

> **注意**: このファイルのパーミッションは 0600（自分のみ読み書き可）ですが、Mac を他人と共有する環境では Keychain による保管への変更を推奨します。

## 設定

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| モデル | gpt-5.4-mini | 翻訳に使用する OpenAI モデル |
| ショートカット | ⌘⇧T | 翻訳トリガーキー（設定画面で変更可） |

## 動作のしくみ

1. ショートカット押下（または メニューバー → 「翻訳」）
2. **AX API** でフロントアプリの選択テキストを直接取得（クリップボードを汚さない）
3. AX 非対応アプリの場合は **Cmd+C 合成**にフォールバック
4. OpenAI Chat Completions API に `stream: true` で送信（temperature 0.2、タイムアウト 60 秒）
5. 訳文をポップアップに頭から逐次表示

## トラブルシューティング

### アクセシビリティ権限を許可したのに動かない

再ビルドのたびにバイナリの署名ハッシュが変わり、macOS が「別アプリ」として扱います。ターミナルで以下を実行し、権限を付与し直してください:

```bash
tccutil reset Accessibility com.personal.HaYaku
```

その後アプリを起動 → システム設定で HaYaku にチェック → アプリを再起動。

### メニューバーアイコンが見えない

Bartender などのメニューバー整理アプリが新しいアイコンを自動的に隠すことがあります。管理画面で HaYaku を表示設定にしてください。

### 特定のアプリでテキストが取れない

- **AX 非対応アプリ**（一部の PDF ビューア等）では Cmd+C フォールバックを自動的に使用します
- それでも失敗する場合は、そのアプリがコピー操作を禁止している可能性があります
- エラーメッセージの `[診断]` 行にどの段階で失敗したかが記録されています

### 変更が反映されない

SPM や TCC のキャッシュが原因のことがあります。クリーンビルドで解消します:

```bash
swift package clean && ./install.sh
```

## 開発者向け

HaYaku は Swift Package Manager の executable ターゲットとして実装し、`.app` バンドルを手動生成しています。アクセシビリティ権限が必要なため、`swift run` ではなく `.app` として起動して動作確認するのが基本です。

### コマンド一覧

| コマンド | 用途 |
|----------|------|
| `swift build` | コンパイル確認（`.app` は生成しない） |
| `./build-app.sh` | リリースビルドしてリポジトリ直下に `HaYaku.app` を生成 |
| `./install.sh` | ビルド → `/Applications` に配置 → TCC リセット → 起動 |
| `swift package clean` | SPM のビルドキャッシュを削除 |

`./install.sh` 実行後は、システム設定で HaYaku のアクセシビリティ権限を再付与してから再起動してください。
